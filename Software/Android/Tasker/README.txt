- Pre requisite -

1) Install Google Now
		https://play.google.com/store/apps/details?id=com.google.android.googlequicksearchbox&hl=it
		
2) Install tasker on your android device
		https://play.google.com/store/apps/details?id=net.dinglisch.android.taskerm&hl=it

3) Install Script Launcher (SL4A)
		https://play.google.com/store/apps/details?id=org.androidideas.scriptlauncher&hl=it

		
- Settings -
Set Google Now as your Home page manager (to enable the "OK GOOGLE" voice command from your home page)
